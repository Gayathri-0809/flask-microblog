# Flask Microblog App 🚀

This is my web application built using **Python** and the **Flask framework**.

---

## 📌 Project Overview

This project is a simple microblog application built using **Flask**.
It demonstrates how to create dynamic web pages using templates and Python.

---

## ⚙️ Features

### ✅ Chapter 1

* Flask project setup
* Virtual environment usage
* Basic routing
* "Hello, World!" web app

### ✅ Chapter 2

* HTML Templates (Jinja2)
* Dynamic data rendering
* Loops for displaying posts
* Template variables
* Separation of logic and UI

## 🚀 How to Run the Project

### 1. Clone the repository

```id="x2d4gu"
git clone https://github.com/Gayathri-0809/flask-microblog.git
cd flask-microblog
```

### 2. Create virtual environment

```id="qchub1"
python -m venv venv
```

### 3. Activate environment

```id="ppl7iy"
venv\Scripts\activate
```

### 4. Install dependencies

```id="4v69dy"
pip install flask
```

### 5. Set environment variable

```id="v2nhfw"
set FLASK_APP=microblog.py
```

### 6. Run the application

```id="q9q6a3"
flask run
```
