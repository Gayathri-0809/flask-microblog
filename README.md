# Flask Microblog App 🚀

This is my first web application built using **Python** and the **Flask framework**.

## 📌 Project Overview

This project demonstrates the basic setup of a Flask application, including routing and running a local development server.

## ⚙️ Features

* Flask project structure
* Virtual environment setup
* Routing using Flask
* Simple "Hello, World!" web page

## 📂 Project Structure

```
microblog/
│── app/
│   ├── __init__.py
│   ├── routes.py
│── microblog.py
│── venv/
│── .gitignore
```

## 🚀 How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/Gayathri-0809/flask-microblog.git
cd flask-microblog
```

### 2. Create virtual environment

```
python -m venv venv
```

### 3. Activate virtual environment

```
venv\Scripts\activate
```

### 4. Install dependencies

```
pip install flask
```

### 5. Set environment variable

```
set FLASK_APP=microblog.py
```

### 6. Run the application

```
flask run
```

## 🌐 Output

Open browser and go to:

```
http://localhost:5000/
```

You will see:

```
Hello, World!
```
